# Руководство по настройке автоматических обновлений

## Шаг 1: Настройка GitHub репозитория

1. Создайте репозиторий на GitHub (например, `your-username/neuro-august`)
2. Откройте файл `updater.py` и замените:
   ```python
   GITHUB_REPO = "your-username/neuro-august"
   ```
   на ваш репозиторий

## Шаг 2: Сборка .exe файла

Используйте PyInstaller для создания исполняемого файла:

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --icon=icon.ico --add-data "web;web" --name "neuro-august" main.py
```

Это создаст файл `neuro-august.exe` в папке `dist/`

## Шаг 3: Создание релиза на GitHub

1. Перейдите в ваш репозиторий на GitHub
2. Нажмите "Releases" → "Create a new release"
3. Заполните поля:
   - **Tag version**: `v1.0.0` (или другая версия)
   - **Release title**: `Version 1.0.0`
   - **Description**: Описание изменений
4. Прикрепите файл `neuro-august.exe` к релизу
5. Нажмите "Publish release"

## Шаг 4: Обновление версии

Когда вы хотите выпустить новую версию:

1. Обновите версию в `version.json`:
   ```json
   {
     "version": "1.0.1"
   }
   ```

2. Обновите версию в `updater.py`:
   ```python
   CURRENT_VERSION = "1.0.1"
   ```

3. Соберите новый .exe файл (см. Шаг 2)

4. Создайте новый релиз на GitHub с новой версией (например, `v1.0.1`)

5. Прикрепите новый .exe файл к релизу

## Как это работает

1. При запуске приложение проверяет GitHub API на наличие новых релизов
2. Если найдена новая версия, показывается уведомление
3. Пользователь нажимает "Обновить"
4. Приложение скачивает новый .exe файл
5. Создается bat-скрипт, который:
   - Ждет 2 секунды
   - Удаляет старый .exe
   - Переименовывает новый .exe
   - Запускает обновленное приложение
   - Удаляет сам себя

## Важные замечания

- Убедитесь, что репозиторий публичный, или используйте GitHub токен для приватных репозиториев
- Версии должны следовать формату semver (например, 1.0.0, 1.0.1, 1.1.0)
- Имя .exe файла в релизе должно заканчиваться на `.exe`
- Приложение проверяет обновления только при запуске

## Пример структуры релиза

```
Tag: v1.0.1
Title: Version 1.0.1
Description:
  ## Что нового
  - Исправлена ошибка с загрузкой диалогов
  - Улучшен интерфейс настроек
  - Добавлена поддержка кириллицы в названиях

Assets:
  - neuro-august.exe (12.5 MB)
```

## Отключение автоматических обновлений

Если вы хотите отключить проверку обновлений, закомментируйте строку в `script.js`:

```javascript
// checkForUpdates();
```
